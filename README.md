# Finance-Data-Analysis
Conducted data cleaning, transformation, and modeling to prepare financial datasets for analysis. Developed interactive dashboards and visualizations in Power BI to track key financial metrics and generate actionable insights for better decision-making.


📊 Finance Data Analysis (Power BI)
This project focuses on analyzing financial and sales data to extract meaningful business insights using Power BI. The dataset was sourced from Kaggle and includes transactional data covering sales, profit, discounts, cost, and customer segments.


🔍 Project Overview
The goal of this project is to perform end-to-end financial analysis by transforming raw data into interactive dashboards and actionable insights. The report helps in understanding revenue patterns, profitability, cost behavior, and segment-wise performance.
⚙️ Key Features & Analysis
Sales & Profit Analysis
Total Sales Amount and Total Profit across all transactions
Year-to-Date (YTD) Sales tracking
Sales contribution by each segment
Monthly Sales Trend for 2023
Product Performance
Top 3 products based on total sales
Top 3 products based on profit
Average manufacturing cost per product
Segment & Customer Insights
Segment with the highest sales
Profit variance between segments
Top-performing segment by profit margin
Average discount given per segment
Geographical Analysis
Total Sales by country (map visualization)
Profit percentage by country
Profit vs COGS comparison by country
Cost & Profitability Insights
COGS (Cost of Goods Sold) trend over time
Impact of discount on profit
Average units sold per transaction
Interactive Dashboard Features
KPI cards (Total Sales, Profit, Units Sold)
Dynamic slicers for filtering data by year and month
Visually rich charts and maps for better decision-making


🛠️ Tools & Technologies
Power BI
Data Cleaning & Transformation (Power Query)
Data Modeling & DAX (Data Analysis Expressions)

📈 Key Insights
Identified high-performing products and segments driving revenue
Analyzed the relationship between discounts and profit margins
Tracked cost trends and their impact on overall profitability
Provided geographical insights to support region-based strategies

🚀 Conclusion
This project demonstrates the use of Power BI to convert raw financial data into meaningful insights through interactive dashboards. It highlights strong skills in data visualization, business intelligence, and financial analysis, enabling data-driven decision-making.
